# 5 Simple Crowdfunding Contract

**Context**

+ People contribute ETH to fund a project.
+ If the goal is met, the owner withdraws.
  - If not, contributors get refunds.

**Requirements:**
+ Owner sets a funding goal and deadline
+ Users can contribute ETH
+ Track each contributor&apos;s amount

**If goal is met:**
- owner can withdraw funds

**If goal is not met after deadline:**
- contributors can claim refunds

Contract must prevent double refunds
